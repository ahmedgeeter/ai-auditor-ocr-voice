# Meridian

**Documents understood. Conversations remembered.**

A production-grade AI Auditor and Intelligence Hub built with React, TypeScript, and Groq.

## Features
- **Auditor Mode**: Real-time document analysis (Llama 3.2 Vision).
- **Voice Mode**: Low-latency voice interaction (Whisper + Llama 3).
- **Premium UI**: Dark charcoal aesthetic with teal accents.

## Tech Stack
- Frontend: Vite, React, Tailwind CSS 4.0
- AI: Groq (Vision, Chat, Whisper)
- Animation: Framer Motion

## Getting Started
1. Clone the repository.
2. `npm install`
3. Add your `VITE_GROQ_API_KEY` to `.env.local`
4. `npm run dev`
